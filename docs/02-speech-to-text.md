# 語音轉文本 (ASR) 服務

## 📋 概述

語音轉文本服務是AI功能的核心組件，將用戶的語音輸入轉換為文本，為後續的AI處理提供基礎。

## 🏗️ 技術架構

### 雲端ASR架構
```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Audio Input   │    │   ASR Service   │    │   Text Output   │
│                 │◄──►│                 │◄──►│                 │
│ - Streaming     │    │ - Preprocessing │    │ - JSON Response │
│ - Buffering     │    │ - Recognition   │    │ - Confidence    │
└─────────────────┘    └─────────────────┘    └─────────────────┘
```

## 🔧 核心功能

### 1. 實時語音識別
- **流式處理**: 支持實時語音流識別
- **多語言支持**: 中文、英文、日文等
- **噪音適應**: 適應不同環境噪音
- **說話人識別**: 支持多說話人場景

### 2. 離線語音識別
- **本地模型**: 支持離線語音識別
- **邊緣計算**: 在設備端進行語音處理
- **隱私保護**: 語音數據本地處理
